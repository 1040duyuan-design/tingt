# TingT Front Copy Rules V1

Judge all front-page copy only by one standard:

- does it feel like TingT himself is online
- or does it feel like the user is trying an AI product

## Must delete

These lines create immediate product-demo feeling:

- `这是基于 TingT 真实聊天风格和经历信息整理出来的数字分身。`
- `这是 TingT 的数字分身网页版本`
- `它不是随机生成的默认 AI 角色`
- any line that explains how the system was distilled, built, tuned, or generated

Reason:
- too much product narration
- user immediately switches into evaluation mode instead of chat mode

## Can keep

These lines still feel close enough to a person being there:

- `TingT`
- `我在。`
- `说吧。`
- `有话直接发。`

Reason:
- short
- no product posture
- no onboarding tone

## Rewrite

Bad:
- `下面直接聊就行。`

Why:
- not terrible, but still sounds like interface instruction

Better:
- `有话直接发。`

Bad:
- `像平时聊天一样发就行。`

Why:
- sounds like usage guidance

Better:
- `想到哪说到哪。`

Bad:
- `聊天`

Why:
- neutral UI label, not wrong, but a bit generic

Better:
- `说`

## Page rule

First screen should only do two things:

1. make the user feel TingT is here
2. let the user send the first line quickly

Do not use the first screen to explain:

- what the product is
- how it was made
- what technology it uses
- why it is special
